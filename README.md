# JLite Compiler CS4212

My submission for NUS's compiler design module assignment.

Example use
```sh
make clean && make
make run FILE=./tests/asg3_tests/asg3_merry_xmas.j OUTPUT=./xmas.s OUT=./xmas
echo 'run executable xmas on gem5'
```
